🧬 Mendel Genetics Pro 

O Mendel Genetics Pro é uma ferramenta de simulação genética desenvolvida em Python, utilizando a biblioteca CustomTkinter para oferecer uma interface moderna e intuitiva. Este projeto foi concebido para facilitar o estudo da 1ª Lei de Mendel, permitindo a visualização prática de cruzamentos genéticos, probabilidades estatísticas e análise de fenótipos/genótipos. 

🚀 Funcionalidades 

Simulação de Cruzamentos: Realiza o cálculo de cruzamentos mendelianos com base em genótipos definidos (Homozigoto Dominante, Heterozigoto, Homozigoto Recessivo). 

Quadrado de Punnett Interativo: Representação visual organizada dos resultados do cruzamento. 

Estatísticas Detalhadas: Cálculo automático de frequências genotípicas e fenotípicas. 

Visualização Gráfica: Geração de gráficos dinâmicos (pizza e barras) utilizando matplotlib para melhor interpretação dos resultados. 

Análise Educacional: Aba dedicada a explicações teóricas sobre o princípio da segregação, hipótese da dominância e probabilidades associadas. 

UI Moderna: Design em modo escuro (Dark Mode) para uma experiência de usuário confortável e focada. 

🛠 Tecnologias Utilizadas 

Python: Linguagem base para a lógica da aplicação. 

CustomTkinter: Framework para construção da interface gráfica moderna. 

Matplotlib: Biblioteca utilizada para a renderização de gráficos estatísticos. 

Tkinter: Utilizado para manipulação de eventos e caixas de diálogo. 

📋 Como Executar 

Certifique-se de ter o Python instalado em sua máquina. 

Clone este repositório: 

Bash 

Instale as dependências necessárias: 

Bash 

pip install customtkinter matplotlib 
 

Execute o arquivo principal: 

Bash 

python main.py 
 

🎯 Objetivo 

Este projeto foi desenvolvido com foco em aplicações educacionais, visando transformar conceitos complexos de genética em experiências visuais interativas e de fácil compreensão.
